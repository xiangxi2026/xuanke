## 你需要准备什么

1. **GitHub 账号**（已有）

## 第一步：推送到 GitHub

1. 访问 https://github.com ，用你的账号登录
2. 点击右上角 "+" → "New repository"
3. 仓库名填：`xuanke`
4. 选 **Public**（公开）
5. 点击 "Create repository"
6. 创建完后，页面上会显示push命令，但先别管

---

## 第二步：我帮你推代码到 GitHub

回到 WorkBuddy（这里），告诉我 **你的 GitHub 仓库地址**，
格式如：`https://github.com/31729799/xuanke`

我来帮你把代码推上去。

---

## 第三步：部署到 Render.com

1. 访问 https://render.com ，点击 "Get Started" 注册（可以用 GitHub 登录）
2. 登录后，点击右上角 "New" → "Web Service"
3. 选择你刚创建的 GitHub 仓库 `xuanke`
4. 配置：
   - Name: `xuanke`
   - Runtime: **Node**
   - Build Command: **留空**
   - Start Command: `node server.js`
   - Free Instance Type: 选 **Free**
5. 点击 "Deploy Web Service"
6. 等待 2-3 分钟，部署完成后你会得到一个网址

---

1. 把部署后得到的网址发给学生/家长
